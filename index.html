202501-deepseek安装

Posted on 2025-02-06 Edited on 2025-02-10 In 开发随笔

下载

下载并安装ollama

安装

安装完成后自动退出安装界面, 并且会隐藏到系统托盘中

在命令行中, 查看ollama命令是否存在

也可以在浏览器中访问http://localhost:11434/来查看ollama是否已经正常运行

此外, 安装完ollama后, 它会开机启动. 通过把启动项删掉的方法, 取消开机启动
C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

搜索你想要安装的模型, 并且下载对应的版本, 我这里16G的4060ti, 最好安装的9G的版本, 通常来说, 如果显存装不下, 那么会占用一定的内存来弥补.

执行命令, 这里run命令是下载并运行, pull命令是下载

默认情况下, 它的模型下载特别慢. 我花了差不多4个小时, 主要是下了一会儿就会停顿住
停顿的话, 可以ctrl+c终止脚本, 然后重新pull, 它会断点续传的,如下图

终于将要下载完毕

下载完毕后, 它会进行sha256校验文件完整性

如果校验通过, 会出现success.

此外还有一个下载模型的方法, 如果你的网络实在是差.可以去阿里的魔搭社区https://www.modelscope.cn/models/下载模型文件.

要注意, 这里选择GUFF格式的模型, 否则ollama不认识, 还需要做额外的模型转换步骤

挑一个量化模型下载,这里的Qxx是各种量化程度, 不过这个网站的下载速度不快, 所以这里选择一个14b的模型(对应9GB显存)

然后在guff的文件同一级目录下, 创建一个配置文件, 比如这里:
ds-14B.txt

# 原始文件从哪里来
FROM ./DeepSeek-R1-Distill-Qwen-14B-Q8_0.gguf    
PARAMETER temperature 0.7                       
PARAMETER top_p 0.95
PARAMETER top_k 40
PARAMETER repeat_penalty 1.1
PARAMETER min_p 0.05
PARAMETER num_ctx 1024                 
PARAMETER num_thread 4                  
PARAMETER num_gpu 8                     


# 设置对话终止符
PARAMETER stop "<｜begin▁of▁sentence｜>"
PARAMETER stop "<｜end▁of▁sentence｜>"
PARAMETER stop "<｜User｜>"
PARAMETER stop "<｜Assistant｜>"


SYSTEM """
"""

TEMPLATE """{{- if .System }}{{ .System }}{{ end }} 
{{- range $i, $_ := .Messages }} 
{{- $last := eq (len (slice $.Messages $i)) 1}}
{{- if eq .Role "user" }}<｜User｜>{{ .Content }}
{{- else if eq .Role "assistant" }}<｜Assistant｜>{{ .Content }}{{- if not $last }}<｜end▁of▁sentence｜>{{- end }}
{{- end }}
{{- if and $last (ne .Role "assistant") }}<｜Assistant｜>{{- end }} 
{{- end }}"""

然后使用这个txt的内容, 创建模型. 

ollama create mymodel-14b -f ./ds-14B.txt

这样, 再使用ollama list的时候, 就有一个mymodel-14b的模型会出现了

运行

通过ollamarun命令运行模型.
ollama run deepseek-r1:32b

会发现, 显存被完全占有, 然后内存占用也很多. 模型的输出有明显卡顿

彻底退出ollama,也就是把任务栏托盘的羊驼退出后, 资源才会被释放, 光光在命令行里结束会话没用.

此外, ollama没用配置, 模型文件又很大. 要怎么才能转移文件位置呢?
ollama的文件位置放在: C:\Users\Administrator\.ollama

可以看到, 这个文件很大, 就是我们的模型文件

把模型移到你想放置的磁盘的位置

然后配置环境变量,win+r打开运行窗口, 输入rundll32 sysdm.cpl,EditEnvironmentVariables

更多环境变量可以参考:

https://github.com/datawhalechina/handy-ollama/blob/main/docs/C2/2.%20Ollama%20%E5%9C%A8%20Windows%20%E4%B8%8B%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E9%85%8D%E7%BD%AE.md

如果要删除模型, 使用:
ollama rm deepseek-r1:32b

事实证明, 16G的显存运行32b的模型, 太卡了. 还是安装14b的模型比较好

界面

然后是在命令行中使用对话不是很美观, 安装open-webui
git clone https://github.com/open-webui/open-webui.git

经过我的各种尝试, 我觉得使用miniconda是最优的(因为在windows下, docker依赖于docker desktop, 而ollama本来就很吃内存了, 再运行一个docker desktop卡死)
去清华源下载miniconda

https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/
